# APKUpdater OSS

*[🇬🇧 Read in English](README_EN.md)*

**APKUpdater OSS** — это инструмент с открытым исходным кодом, который упрощает процесс поиска обновлений для установленных приложений. Он работает аналогично магазину приложений, но собирает обновления из множества независимых источников.

Этот проект является глубоко модифицированным и современным форком оригинального [APKUpdater](https://github.com/rumboalla/apkupdater), созданного [rumboalla](https://github.com/rumboalla). Огромное спасибо оригинальному автору за великолепную основу!

## 🌟 Что нового

* **Интеграция с RuStore** — загрузка APK и changelog напрямую из официального российского магазина.
* **Поддержка пользовательских Git-репозиториев** — добавляйте проекты с GitHub/GitLab, настраивайте пакет, regex для APK и получайте обновления наряду со стандартными источниками.
* **Редизайн на Material Design 3** — обновлённая навигация, карточки, типографика, плавные переходы и поддержка динамических цветов.
* **Безопасная смена пакета** — приложение устанавливается как `com.apkupdateross`, не конфликтуя с оригинальным APKUpdater.

## ✨ Основные возможности
* **Источники обновлений**: **APKMirror**, **Aptoide**, **F-Droid**, **IzzyOnDroid**, **APKPure**, **GitLab**, **GitHub**, **Google Play**, **RuStore** и ваши собственные Git-репозитории.
* **Поиск приложений**: быстрый поиск по поддерживаемым площадкам.
* **Автопроверки**: расписание фоновых проверок с уведомлениями и настройкой интервала (1/3/7 дней).
* **Режимы установки**: обычный Session Installer, Root и Shizuku.
* **Интерфейс для Android TV** и поддержка Android 7.0 (24) – Android 14 (34).
* **Темы**: тёмная, светлая, системная, а также динамические цвета Android 12+.
* **Прямая и root-установка** APK без рекламы и трекеров.

## 📥 Скачать
Перейдите в раздел **Releases** этого репозитория, чтобы загрузить последнюю стабильную версию (`com.apkupdateross-release.apk`).

## 💻 Сборка из исходников

```bash
./gradlew assembleRelease
```

Debug-сборку можно получить командой `./gradlew assembleDebug`.
